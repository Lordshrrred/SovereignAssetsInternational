# GLOBAL Web Drop-In Assets

Use these when the browser-rendered stylized A in `GLOBAL` does not match the logo.

## Recommended Site Usage
- Use `svg/clean-stylized-a-full-color.svg` when only the custom A needs to be dropped into a layout.
- Use `svg/clean-global-wordmark-full-color.svg` when the whole word `GLOBAL` should render as one consistent asset.
- Use the `global-wordmark-*` and `stylized-a-*` files if you want traced versions pulled directly from the supplied logo photo.
- White, blue, and black variants are included for dark/light sections.

## HTML Example
```html
<img src="/brand/global-wordmark-full-color.svg" alt="GLOBAL" />
```

All files have transparent backgrounds. PNG exports are high resolution. The `clean-*` SVG files are hand-vectorized for crisp website rendering.
